Installation de GesLyCo-X
Comment installer cette application ?

1. Cliquez sur le bouton "Télécharger" pour lancer le téléchargement du programme d'installation de l'application GesLyCo-X (fichier INSTALL.EXE) .
(selon le navigateur, une confirmation pourra vous être demandée).

2. Une fois le téléchargement effectué, exécutez le programme.

3. Suivez les instructions données par le programme d'installation de l'application.

Remarque : Vous pouvez directement exécuter l'installation après son téléchargement avec le choix "Exécuter" du navigateur.
